空调语音控制系统（语音识别 + NLP 意图分类 + 温度模式回归预测）

2024.12 – 2025.03

使用 Whisper 构建 ASR 模块，实现家庭场景下口语化空调指令的语音转文字；通过噪声增强与 VAD 预处理提升识别鲁棒性。

基于 BERT搭建双任务 NLP 模型，对 ASR 文本进行 Tokenization、Embedding、上下文语义编码（Contextual Representation）

	1.意图分类模型（Cross-Entropy Loss）：识别开/关机、调温、模式切换、风速调节等指令意图。
	
	2.温度/模式回归模型（MSE / SmoothL1 Loss）：预测温度调节幅度（如“冷一点”“升两度”）并解析目标温度与目标模式

采用 Hybrid NLP（规则模板 + BERT 表示） 处理弱语义与隐式表达，如“有点热”“再高一点”，结合同义词扩展与文本正则化提升解析效果

将模型输出转换为结构化控制参数（如 `power=on`、`mode=cool`、`delta_temp=+2`、`target_temp=26`），实现端到端语音控制指令生成
