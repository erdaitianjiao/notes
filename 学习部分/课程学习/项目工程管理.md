## 项目工程管理总结

### 一、概述

#### (1) 项目概述

- 项目定义

项目是在一定的时间、资源、环境等约束条件下，为了实现特定的目标所完成的一次性任务或付出的努力。

- 项目特征

  1. 一次性
  2. 独特性
  3. 资 源 制 约 性
  4. 组织的临时性和开放性
  5. 后果的不可挽回性

- 项目生命周期

  | 名称     | 主要内容                                                     |
  | -------- | ------------------------------------------------------------ |
  | 启动阶段 | 确定需求目标，项目立项，可行性研究，项目批准，建立项目组织，确定项目经理等 |
  | 规划阶段 | 预算和进程制定，初步设计，详细设计，项目执行准备等           |
  | 执行阶段 | 项目实施，项目控制等                                         |
  | 收尾阶段 | 评价，总结项目目标的完成程度，项目验收，文档整理，项目交接等 |

- 项目里程碑

  里程碑（Milestone），项目中的重大事件，通常指一个主要可交付成果的完成。它是项目进程中的一些重要标记，是在规划阶段应该重点考虑的关键点，里程碑既不占用时间也不消耗资源。

- 可交付成果

  可交付成果( Deliverable )是指某种可以核实的工作成果或事项。一般来说，项 目 有阶段可交付成果和最终可交付成果两种形式。

#### (2) 项目管理概述

- 项目管理的含义

  项目管理是以项目及其资源为对象，运用系统的理论和方法对项目进行高效率的计划、组织、实施和控制，以实现项目目标的管理方法体系。

- 项目管理的特征

  1. 项目管理具有创新性
  2. 项目管理是一项复杂的工作，具有较强的不确定性
  3. 项目管理需要专门的组织和团队
  4. 项目经理的租用非常重要

- 项目管理的要素

  进度、成本、质量   (项目范围 目标)

  成功的项目必须满足客户、管理层和供应商在时间、费用和性能上的不同要求

- 项目管理的工作过程

  1. 项目启动
  2. 项目规划
  3. 项目执行
  4. 项目控制
  5. 项目收尾

- 项目生命周期的阶段和管理工作过程的关系

  1. 差异一

     - 项目管理过程(启动过程、规划过程、执行过程、控制过程和收尾过程)的名称和典型项目生命期阶段(启动阶段、规划阶段、执行阶段和收尾阶段)名称存在相同部分，但是它们的含义却有差异。
     - 项目生命期所包括的四个阶段没有重复，是从项目实现过程的角度考虑。

  2. 差异二

     - 项目管理的五个工作过程并不是独立的---次性过程，它贯穿于项目生命期的每--个阶段。
     - 项目的任何一个阶段都包含一个或几个“启动一规划一执行一控制一收尾”的管理工作过程

### 二、项目运行环境和组织结构

#### (1) 项目运行环境

项目所处的环境可能对项目的开展产生有利或不利的影响，这些影响主要源自事业环境因素和组织过程资产两个方面。

- 事业环境因素

  事业环境因素是指项目团队无法控制、并对项目产生影响、限制等约束条件，这些约束条件可能来自组织的内部或外部。

  - 分类

    组织内部的事业环境因素
    
    1. 组织文化、结构和治理
    2. 设施和资源的地理分布
    3. 基础设施
    4. 信息技术软件
    5. 资源可用性
    6. 员工能力
    
    组织外部的事业环境因素
    
    1. 市场条件
    2. 社会和文化影响和问题
    3. 法律限制
    4. 商业数据库
    5. 政府或行业标商业数据库准
    6. 财务考虑因素
    7. 物理环境因素

  正是因为项目团队自身对事业环境因素无法控制的原因，对于事业环境因素而言，项目团队只能是采取遵守、被动接受、入乡随俗的态度

- 组织过程资产

  组织过程资产是项目执行组织所特有的，并在项目管理过程中积累和应用的计划、过程、政策、程序和知识库等资源。

  组织过程资产并非局限于一个项目或一个组织，。任何一类用于影响项目成功的资源均可被视为组织过程资产。

  它是项目执行组织在项目操作过程中积累或应用的无形资产
  
  - 分类
  
    1. 过程、政策和程序
  
       第一类组织过程资产的更新通常不是项目工作的内容，而是由项目管理办公室或项目以外的其他职能部门完成的。
  
    2. 组织知识库
    
       第二类组织过程资产是在项目生命周期结合项目信息而更新的
  
- 事业环境因素与组织过程资产
  
  对于项目团队来说，虽然同样都是作为项目管理工作过程的输入，前者只能被动接受，后者则可以主动参与。
  
  
  

#### (2) 项目组织结构

eg. 三省六部制

- 定义

  1. 组织是由两个以上的人组成，为了实现一个共同目标，以特定的形式、加以编制的集合体。
  2. 项目组织是为完成特定的项目任务而建立起来的，从事项目具体工作的载体。项目组织同一般的组织一样，具有相应的领导(项目经理)、规章制度(项目章程)、相关人员(项目团队)及组织文化等。项目组织的突出特点是临时性和灵活性

  采用的将若干个个体组合为部门、然后将部门再组合成整个组织的方式

  是确保跨部门的沟通，协作和力量整合的制度设计
